# Booking System Modern SaaS

Sistem booking modern dengan arsitektur REST API.

## Tech Stack
- **Frontend**: React, TailwindCSS, Axios, React Router, React Query
- **Backend**: Golang (Fiber), GORM, PostgreSQL, JWT
- **Design**: Modern SaaS aesthetic, responsive, glassmorphism

## Setup Manual

### Backend
1. Masuk ke folder `backend/`
2. Atur configuration di `.env`
3. Jalankan `go run main.go`

### Frontend
1. Masuk ke folder `frontend/`
2. Jalankan `npm install`
3. Jalankan `npm run dev`

### Fitur
- Role-based Access Control (Admin & Customer)
- Realtime Booking Availability
- Modern Dashboard (User & Admin)
- Responsive Design
- Clean REST API Architecture
